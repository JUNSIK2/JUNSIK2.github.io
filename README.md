# 오르미 1차 프론트엔드 프로젝트

## 1. 프로젝트 목표 및 기능

### 1.1 프로젝트 목표
* 제공된 템플릿과 명세서를 활용해 웹페이지를 작성한다.
* 프론트엔드 기술을 습득하여 웹개발자가 되기 위한 능력을 키운다.

### 1.2 프로젝트 주요 기능
* 탑 스크롤 버튼 구현, 스크롤이 최상단에 위치 시 버튼 감추기
* 이메일 유효성을 검사, 모달 창 구현
* 사이드 메뉴 바 구현
* 카카오맵 API 사용
